# Rahul Anilkumar Portfolio

A personal developer portfolio built with Laravel, Blade, Bootstrap, JavaScript, and Docker.

## Features
- Home, About, Projects, Contact pages
- Contact form with email functionality
- Responsive UI
- Laravel backend routing
- Docker deployment support

## Tech Stack
Laravel, PHP, Blade, Bootstrap, JavaScript, Docker, Render
